# Basic-Question-of-interview-documentation

PART 1 — HTML (20 Questions)

Semantic Tags (4 Questions)
Q1: What are semantic tags in HTML?
Semantic tags are HTML elements that clearly describe their meaning in both the browser and developer understanding. Examples include <header>, <footer>, <article>, and <section>.

Q2: Why are semantic tags important?
Semantic tags improve readability, accessibility, SEO, and help screen readers understand webpage content structure.

Q3: What is the difference between <div> and semantic tags?
<div> is non-semantic and only groups content, while semantic tags like <section> or <article> describe the role and meaning of the content.

Q4: Name five commonly used semantic tags.
Header, Nav, Main, Section, Article, Footer.

Attributes (4 Questions)
Q5: What is an HTML attribute?
Attributes provide additional information about HTML elements such as id, class, src, href, etc.

Q6: What does the href attribute do?
The href attribute defines the destination URL of an anchor link.

Q7: What is the purpose of the alt attribute in images?
It provides alternative text for images, improves accessibility, and helps SEO.
Q8: What is the difference between id and class attributes?
 id is unique for one element, while class can be reused for multiple elements.
HTML Elements (4 Questions)
Q9: What is an HTML element?
 An HTML element is a combination of a start tag, content, and an end tag.
Q10: What is the difference between block and inline elements?
 Block elements take full width and start on a new line (like div, p) while inline elements do not break line and occupy only required width (like span, a).
Q11: What is the root element of an HTML document?
 The <html> element.
Q12: What does the <head> element contain?
 Metadata, title, scripts, CSS links, SEO data, etc.


Forms and Inputs (4 Questions)
Q13: What is an HTML form?
 An HTML form is used to collect user input using <form> element.
Q14: What is the purpose of the action attribute in a form?
 It defines where form data will be sent after submission.
Q15: Name four different form input types.
 Text, Email, Password, Checkbox, Radio, File.
Q16: What does the placeholder attribute do?
 It displays a short hint about expected input inside the field.
Media (4 Questions)
Q17: How to embed an image in HTML?
 Using <img src="" alt="">.
Q18: Which tag is used to play video in HTML?
 <video controls src=""></video>.
Q19: Which tag is used to play audio?
 <audio controls src=""></audio>.
Q20: What is the purpose of the controls attribute?
 It shows play, pause, volume control buttons.

PART 2 — CSS (15 Questions)
Selectors (3 Questions)
Q21: What are CSS selectors?
 Selectors are used to target HTML elements to apply styles.
Q22: What is the difference between class selector and id selector?
 Class selector starts with . and can be reused, id selector starts with # and must be unique.
Q23: What is a universal selector?
 * applies styles to all elements.
Box Model (3 Questions)
Q24: What is the CSS Box Model?
 It consists of content, padding, border, and margin.
Q25: What does padding do?
 It creates space inside the element between content and border.
Q26: What does margin do?
 It creates space outside the element between other elements.

Positioning and Layout (3 Questions)
Q27: Name five CSS positioning values.
 Static, Relative, Absolute, Fixed, Sticky.
Q28: What does position absolute do?
 Positions an element relative to its closest positioned parent.
Q29: What is Flexbox used for?
 For aligning and distributing space in a layout efficiently.
Responsive Design (3 Questions)
Q30: What is responsive web design?
 Designing websites that adapt to different screen sizes.
Q31: What are media queries?
 CSS rules that apply styles based on device width and height.
Q32: Why is responsiveness important?
 Improves user experience, accessibility, and mobile support.
Styling (3 Questions)
Q33: How to change text color in CSS?
 Using color property.
Q34: How to change background color?
 Using background-color.
Q35: How to change font size?
 Using font-size.

PART 3 — JavaScript (25 Questions)
DOM Manipulation (5 Questions)
Q36: What is the DOM?
 DOM is the Document Object Model that represents HTML structure in a tree format.
Q37: How to select an element by id?
 document.getElementById("id").
Q38: How to select elements by class?
 document.getElementsByClassName("class").
Q39: How to change the text of an element?
 element.innerText = "Hello".
Q40: How to change CSS style with JS?
 element.style.color = "red".
Control Flow (5 Questions)
Q41: What is control flow?
 The order in which JavaScript code executes.
Q42: What is an if statement?
 Executes code only if a condition is true.
Q43: What is else?
 Executes when the condition is false.
Q44: What is a loop?
 Repeat a block of code multiple times.
Q45: Name two loops in JS.
 For loop, While loop.

 String with backticks supporting ${variables}.

 JavaScript Object Notation for data exchange.
